    Centos7 git安装

    一.安装

    • 检查是否安装过:rpm -qa | grep git
    • 安装git:yum install git
    • git –version

    二.配置ssh key

    • 初始化ssk-key，回车
    • eval “$(ssh-agent -s)”
    • ssh-add ~/.ssh/id_rsa

      在网站添加你的ssk-key

    • /root/.ssh/id_rsa.pub

      测试连接

    • ssh -T git@github.com

    使用Gogs搭建git服务器

    1.配置Gogs所需的环境

    • 安装nginx
    • 安装git
    • 安装MySQL
    • 为Gogs创建单独的用户

    2.配置与运行Gogs

    • 修改Gogs service配置文件
    • 切换到root账户然后复制到/etc/init.d/
    • 增加执行权限
    • 复制service
    • 启动Gogs

    3.浏览器配置gogs

    • 打开浏览器3000端口
    • 配置gogs.相关资料：gogs配置手册
    • gogs配置文件：/home/git/gogs/custom/conf/app.ini
