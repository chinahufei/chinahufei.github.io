IDEA使用快捷键

  • 代码自动提示
  • 打印-sout
  • 局部变量-.var
  • 成员变量-.field
  • 格式化字符串-.format
  • 判断非空-.nn,.null,.notnull
  • 取反-.not
  • 遍历-.for,.fori,.forr
  • 返回值-.return
  • 同步锁-.synchronized
  • Lambda表达式-.lamda

IDEA开发需要装的插件

  • Scala
  • Lombok(不用写实体类的get、set方法)

IDEA激活方法

  • Help–>Edit Custom VM Option –>添加一下内容
  • Help -> Register–>填入验证码
  • 然后重启就可以使用了
