Hexo搭建个人博客
发表于 2019-12-06| 更新于 2020-01-06|工具
阅读量:

基本流程

1.安装Git
2.安装Node.js
3.安装Hexo
4.GitHub创建个人仓库
5.生成SSH添加到GitHub
6.将hexo部署到GitHub
7.设置个人域名
8.发布文章

安装git，可以参考git安装那篇博客。

NodeJs安装，可以参考网上博客，node -v 和 npm -v 有返回值即可。

安装Hexo

node_modules: 依赖包
public：存放生成的页面
scaffolds：生成文章的一些模板
source：用来存放你的文章
themes：主题
** _config.yml: 博客的配置文件**

github注册账户，参考https://my.oschina.net/chinahufei/blog/1577979

生成SSH添加到GitHub

将hexo部署到GitHub

  • 配置_config.yml文件

    npm install hexo-deployer-git –save
    hexo clean
    hexo generate
    hexo deploy

设置个人域名

  • 在域名服务商购买域名
  • 在域名服务商账户做解析，解析到github对应的github page
  • 在github page做域名绑定。(登录GitHub，进入之前创建的仓库，点击settings，设置Custom domain，输入你的域名)
  • 也可以注册一个国内的coding，实现双域名解析，国内解析coding,国外解析github。不过这个需要域名服务商支持国内外双线路解析。
