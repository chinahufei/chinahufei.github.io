    版本

    hadoop: 2.6.0
    cdh: 5.12.0
    mysql: 5.7.11
    hive: 1.1.0
    scala: 2.12.0
    spark: 2.4.4

    基本步骤

    1.安装包准备
    2.安装JDK
    3.安装mysql
    4.hadoop部署
    5.hive部署
    6.客户端连接hive

    cloudera官网下载hadoop,hive,hbase安装包

    • 在官网下载以cdh5.12结尾的hadoop,hive安装包, 地址：http://archive.cloudera.com/cdh5/cdh/5/
    • 下载JDK, 下载mysql5.11.7,mysql连接jar包
    • 准备dbeaver的hadoop-common，hive-jdbc-standalone的jar包

    安装JDK

    安装mysql

    • 参考: https://blog.chinahufei.com/2019/12/02/%E7%A6%BB%E7%BA%BF%E5%AE%89%E8%A3%85mysql/

    hadoop部署

    配置.bashrc

    • source .bashrc
    • which hdfs
    • which hive

    配置信任关系

    • ssh-keygen -t rsa -P ‘’ -f ~/.ssh/id_rsa
    • cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
    • chmod 0600 ~/.ssh/authorized_keys
    • ssh hadoop001 date

    配置文件及 NN SNN DN RM NM都以hadoop001启动

    • vi core-site.xml
    • vi hdfs-site.xml
    • vi slaves
    • vi mapred-site.xml
    • vi yarn-site.xml

    格式化启动

    • hdfs namenode -format
    • start-dfs.sh
    • start-yarn.sh
    • jps

    打开网址

    hive部署

    部署mysql及赋予权限

    • grant all privileges on . to hufei@’%’ identified by ‘password’;
    • flush privileges;

    配置hive-site.xml

    启动metastore + hiveserver2服务

    • nohup hive –service metastore > ~/log/metastore.log 2>&1 &
    • nohup hiveserver2 > ~/log/hiveserver2.log 2>&1 &

    测试hiveserver2服务是否ok

    • 第一种：beeline
    • 第二种：hive

    客户端连接hive

    • 下载dbeaver
    • 点击连接，导入hadoop-common，hive-jdbc-standalone的jar包
    • 图形化界面操作书写hive sql

    安装scala

    • 切换到root用户(su)
    • 解压scala安装包到/usr/local
    • 添加环境变量 vim /etc/profile
    • 配置立即生效(source /etc/profile)

    安装spark

    • 切换到hufei用户(su - hufei)
    • 解压安装包
    • 创建软连接
    • 配置spark
    • 配置环境变量(vim ~/.bashrc)
    • 立即生效(source ~/.bashrc)
    • 启动spark
    • 访问8080端口

    安装azakaban

    • 下载安装包
    • 解压
    • 手动配置gradle-4.6-all.zip
    • 在根目录下执行编译
    • 安装solo-server
    • 打开webUI,修改配置在conf中
